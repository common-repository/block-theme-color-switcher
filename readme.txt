== Description ==

Block Theme Color Switcher plugin allows users to select a color palette for a Block Theme from frontend. The plugin integrates seamlessly with WordPress, offering a user-friendly interface where visitors can effortlessly switch between different color schemes. This feature enhances the visual experience of a website
